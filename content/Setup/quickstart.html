3.2. Quick start

You can download our source code from Github, and:

  • if you downloaded it to a server, put the folder into shiny app directory. If you hadn't install the shiny server, see here.
  • if you download to a PC, you can use the cd command of the shell into our source code root path, then open R. R code below shows how to temporary run a shiny app. For more information, see here or here.
library(shiny)
runApp()

Our source on Github already have a test data set. If the system environment is well configurated, you could already get start with it.

3.2.1. Environment request

To run the SnpHub, make sure these softwares are already installed:

  • samtools
  • bcftools
  • seqkit
  • tabix

Alse, these R packages are also needed:

  • ggplot2
  • ggmap
  • dplyr
  • rjson
  • shiny
  • pegas
  • vcfR
  • ape
  • DT
